# Bileton

Description of a console application for booking and buying movie tickets  

Our console application, developed in Java programming language, provides users with a convenient and functional tool for planning trips to the movies. The main goal of the application is to simplify the process of selecting, booking and buying tickets by offering a full range of features in an intuitive way.  

The main functions of the application are:  

1. View available sessions:  
   The user can view movie schedules, movie titles, genres, duration, age restrictions and other details. This allows one to make an informed choice before buying a ticket.  

2. Selecting seats in the hall:  
   The application provides an interactive display of the auditorium layout. The user can select seats based on their location, availability and preferences.  

3. Pre-booking:  
   If you want to book a ticket in advance, the app allows you to do so. The selected seat will be temporarily assigned to you so that you can complete the process later.  

4. Ticket Payment:  
   The app supports various payment methods including bank cards and e-wallets. The payment process is secure and takes minimal amount of time.  

5. Receiving confirmation:  
   After successful payment, the user receives a booking confirmation with a unique ticket ID. This code can be used to enter the hall.  

6. Order history:  
   For the convenience of users, the application saves information about previously purchased tickets. This helps in keeping track of the sessions attended and facilitates repeat bookings.  

Benefits of the app:  

- Convenience: A concise and clear interface allows users to quickly learn the app, even without prior experience with similar systems.  
- Mobility: You can book anywhere the console is available and not worry about your favorite seats being occupied.  
- Efficiency: Save time by automating the ticket search, selection and payment processes.  

Technical Features:  

The application is built using advanced features of the Java language. It has a modular architecture that allows easy adaptation and expansion of functionality. A relational database is used for data storage, which ensures reliability and availability of information.  

Our console application will become your indispensable assistant in planning your leisure time and guarantees comfortable booking of tickets for any movie show.
